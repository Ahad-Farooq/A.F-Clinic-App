A.F Clinic - Management System
A.F Clinic is a modern management portal designed to streamline healthcare operations between staff and doctors. The system ensures organized patient registration, live queue management, and secure medical archiving.

Key Features
Receptionist Portal
Easy Registration: Quickly register patients and assign them to specific specialists.

Live Patient Queue: Monitor the active patient list in real-time.
Doctor Roster: Automatically view each doctor’s specific shifts, room numbers, and availability days.
Medical Archives: Treated patients are automatically moved from the live queue to the "Patient Records" history.

🩺 Doctor Portal
Assigned Appointments: Doctors only see patients specifically booked for them.
Smart Diagnosis: The system provides health insights and risk analysis based on patient age and gender.
Digital Prescription: Doctors can write prescriptions and advice directly into the system to finalize consultations.

🛠️ Tech Stack
Frontend: React.js & Tailwind CSS
Backend: Node.js & Express.js
Database: MongoDB (Mongoose)
Engine: Google Gemini Pro (Smart Diagnosis)
